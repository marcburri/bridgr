# Summarize a Mixed-Frequency Model

Summarize a Mixed-Frequency Model

``` r
# S3 method for class 'mf_model'
summary(object, ...)
```

## Arguments

- object:

  A `"mf_model"` object returned by
  [`mf_model()`](https://marcburri.github.io/bridgr/reference/mf_model.md).

- ...:

  Unused.

## Value

`object`, invisibly.
